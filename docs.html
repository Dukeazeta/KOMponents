Installation

Add KOMponents to your Flutter project by adding the following to your pubspec.yaml:

dependencies:
  komponents: ^1.0.0

Then run:

flutter pub get

Basic Usage

Import KOMponents in your Dart file:

import 'package:komponents/komponents.dart';

Use components in your widgets:

KButton(
  text: 'Click Me',
  onPressed: () {
    print('Button clicked!');
  },
)

Customization

KOMponents are highly customizable. Here's how to customize the appearance:

KButton(
  text: 'Custom Button',
  style: KButtonStyle(
    backgroundColor: Colors.blue,
    textColor: Colors.white,
    borderRadius: 8.0,
  ),
)
